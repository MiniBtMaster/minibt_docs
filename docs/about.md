
# 关于 MiniBT

## 项目简介

**MiniBT** 是一个专为量化交易爱好者设计的轻量级、高性能的回测框架。我们的目标是让量化交易策略开发变得更加简单、高效。

### 核心理念

- **简单易用**: 提供直观的 API 设计，降低学习成本
- **功能强大**: 涵盖从数据获取到策略回测的全流程
- **性能优异**: 优化的计算引擎，支持大规模数据回测
- **开源免费**: 遵循 MIT 协议，完全开源免费

## 主要特性

### 🚀 快速上手
```python
from minibt import *

class MyStrategy(Strategy):
    def next(self):
        if self.data.close.new > self.data.close.new:
            self.data.buy()
```

### 📊 丰富指标

内置 300+ 常用技术指标，支持自定义指标开发：

- 趋势指标（MA, MACD, Bollinger Bands）
- 动量指标（RSI, Stochastic, CCI）
- 波动率指标（ATR, Volatility）
- 成交量指标（OBV, Volume Profile）

### 🔧 参数优化

内置智能参数优化系统：

```python
bt = Bt()
bt.optstrategy(['profit', 'sharpe'],(1., 1.), 
                opconfig=OptunaConfig(n_trials=100, logging=False), 
                op_method='optuna', 
                show_bar=True,
                len1=(5, 10, 1), 
                len2=(11, 20, 1))
bt.run()
```

### 📈 可视化分析

丰富的回测结果可视化：

- 资金曲线
- 持仓分析
- 交易记录
- 绩效指标

## 技术架构

### 核心组件

- **KLine**: 统一数据接口，支持多种数据源
- **Strategy**: 策略基类，提供完整的交易接口
- **Indicator**: 指标系统，支持链式计算
- **Backtest**: 回测引擎，支持多时间框架

### 数据流架构

```
数据源 → KLine → 策略引擎 → 交易执行 → 绩效分析 → 可视化
```
